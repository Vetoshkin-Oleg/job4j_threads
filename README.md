# job4j_threads  

В курсе производилась работа с ветками git.
До этого момента мы проектировали и работали только с однопоточными приложениями. В разделе "Multithreading" мы познакомимся с многопоточностью в Java. Многопоточные приложения намного производительные, чем однопоточные, так как позволяют выполнять несколько задач одновременно, то есть делить задачи между потоками (нитям). Многопоточность - это сложный механизм, незнание которого может привести к различным негативным последствиям при работе в многопоточной среде.
